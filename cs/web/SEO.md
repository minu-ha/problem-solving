SEO란 뭘까...
==========

    * SEO가 뭐임?
      -> 검색엔진 최적화임

    * 왜 필요한거임?
      -> 웹사이트의 서비스나 마케팅적인 부분은 사용자의 유입이 중요한데 유입을 높이기 위해선 검색과 노출이 잘 돼야함

    * 어떻게 작동하는거임 ?
      -> 검색 엔진 봇들은 사이트의 데이터를 HTML파일에서 크롤링함
         CSR 방식은 요청시 HTML이 비어있기 떄문에 검색 엔진에 노출되기 어려움
         SSR은 서버에서 컨텐츠가 포함된 HTML을 제공하기 때문에 검색 엔진에 수집되기 쉬음
